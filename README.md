# Evaluating the Utility of Synthetic Longitudinal Data Using the `synthpop` package

This repository host a project demonstrating the evaluation of synthetic data utility of a complex longitudinal data with multi-level structure.

## Project Highlights
-  Synthetic data generation using the `synthpop` package
-  Parallel processing with the `furrr` package
-  Utility evaluation using measurement metrics `pMSE` and standardized form `S_pMSE`
-  Complex longitudinal, multi-site/hierarchical data structure
